# HUMSJ Information Technology Sector

## Project Overview

**HUMSJ IT Management System** - A comprehensive platform for managing Haramaya University Muslim Student Jama'a's IT operations, member engagement, and community activities.

## About HUMSJ

The Haramaya University Muslim Student Jama'a (HUMSJ) Information Technology Sector is dedicated to:
- Digitizing Islamic student community operations
- Enhancing member engagement through technology
- Streamlining communication and event management
- Strengthening our Islamic community through innovation

## Features

- **Member Management**: Complete member lifecycle with role-based permissions
- **Event Management**: Comprehensive event system with registration and attendance tracking
- **Islamic Content Hub**: Digital library with Quran, Hadith, and educational resources
- **Communication Platform**: Multi-channel messaging and announcements
- **Analytics Dashboard**: Real-time insights and reporting
- **Prayer Times Integration**: Automated prayer time calculations and reminders

## How to Run This Project

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Development Setup

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd humsj-it-sector

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

### Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint for code quality

## Technology Stack

This project is built with modern web technologies:

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with shadcn/ui components
- **3D Graphics**: Three.js for immersive animations
- **Backend**: Supabase for database and authentication
- **Build Tool**: Vite for fast development and building
- **Deployment**: Optimized for modern web hosting
